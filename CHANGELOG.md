# Changelog

All notable changes to the FabricFlow Shared Library will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Repository structure for version-controlled tasks and workflows
- Metadata schemas for tasks and workflows
- Manifest system for programmatic discovery
- Automated validation scripts
- GitHub Actions CI/CD workflows
- Comprehensive documentation (CONTRIBUTING.md, INTEGRATION_GUIDE.md)
- Integration examples for CLI and Compute Fabric

## [1.0.0] - 2026-02-10

### Added
- Initial release of FabricFlow Shared Library
- Core task categories: array, audio, camera, diffusion, document, embedding, huggingface, image, io, llm, onnx, text, vectorstore, video, vision, web, workflow
- Sample workflows: YOLO object detection, stable diffusion, document scanning, camera background blur, LLaMA fine-tuning
- Basic README and CODEOWNERS configuration
